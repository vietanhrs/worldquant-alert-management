# WorldQuant Alert Management

Interview-oriented SPA project for a WorldQuant internal alert-management and
incident-collaboration product.

## Tech Stack

- React
- TypeScript
- Bun
- Vite
- Tailwind CSS
- Mantine

## Current Milestone

This step focuses on UI/UX discovery, product thinking, information
architecture, user flows and screen-level wireframe strategy. The first real
artifact is the `screen/` folder. UI implementation will come later.

## Commands

```bash
bun install
bun dev
bun run build
bun run lint
```
